Control pages

Control pages

Control pages from each subsystem serve to provide an interface to change settings, start measurements, or read out values of devices used in that subsystem. Each control page is defined separately for each subsystem/database. For more information on how to construct control pages, see the tutorial.

The controls are always available at:

http://db.nedm1/page/control/nedm/[sub_system_name]
